
# My Free Tunes

Free music platform app designed to provide users with a seamless and enjoyable music streaming experience without any subscription fees.


## Features

- Listen and upload song
- Create album
- Playlists
- Playlists collaboration
- Cross platform
- Offline mode
- Playback speed

## Roadmap

- Comunity post
- Lyrics
- Audio effects
- Light/dark mode
- Language options


## Installation

Install dependencies with npm. Run

```bash
  npm install
```
Build the app

```bash
  npm run build
```

Start the server

```bash
  npm run start
```
